| Variable | Description                                                                      |
|---|----------------------------------------------------------------------------------|
| `DSXCONNECTOR_ASSET` | Root asset location (bucket, path, etc.).                                        | 
| `DSXCONNECTOR_FILTER` | Optional rsync-style include/exclude rules.                                      | 
| `DSXCONNECTOR_ITEM_ACTION` | Action taken when malicious files are detected (nothing, delete, move, move_tag) | 
| `DSXCONNECTOR_ITEM_ACTION_MOVE_METAINFO` | If move or move_tag, destination for moved objects.                              |
